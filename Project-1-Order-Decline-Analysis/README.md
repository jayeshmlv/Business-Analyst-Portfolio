# Order Decline Analysis | Business Analyst Case Study

## Overview

This Business Analysis case study investigates a 20% decline in orders for a food delivery platform operating in Indore.

The objective was to identify root causes behind the decline, assess business impact, and recommend actionable solutions to improve order volume and customer retention.

## Business Analysis Activities

* Stakeholder Analysis
* Root Cause Analysis (RCA)
* Fishbone Diagram
* Power-Interest Matrix
* Process Analysis
* Business Recommendations

## Skills Demonstrated

* Requirement Gathering
* Stakeholder Management
* Business Process Analysis
* Root Cause Analysis
* Problem Solving
* Business Documentation
